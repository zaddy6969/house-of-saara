## Included

- Editorial ivory visual system
- Bangalore and Lavelam location selector
- Wedding, private celebration and corporate catering sections
- Interactive menu concepts
- Intentionally empty Vegetarian collection for later curation
- Event brief planner with copy-to-clipboard
- Responsive mobile navigation
- Accessible semantic HTML and reduced-motion support
- SEO-ready metadata

## Run locally

Open `index.html` directly, or serve the folder:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

The project is a static site and can be deployed directly on Vercel with no build command. Set the framework preset to **Other** and leave the output directory blank.
